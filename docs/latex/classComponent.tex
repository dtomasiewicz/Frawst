\subsection{Detailed Description}
Base \hyperlink{classComponent}{Component} class for the \hyperlink{classFrawst}{Frawst} framework.

Components are modular extensions of controller (business) logic. Components are instantiated on-\/demand and each controller instance has its own set of components.
