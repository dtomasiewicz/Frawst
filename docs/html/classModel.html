Sets the value of a model property or relation. You can flag autoRelate to false to avoid setting any relations, if your data is compromisable (for example, if you are simply set()ing form data, users may be able to cross-site-script to change the relations).

Returns:
true if everything set correctly, false if any property failed to set. since primary keys are ignored (they should be treated as immutable) a primary key may fail to set but still return true.
