static Loader::importPath ( $  class,
$  scope = null 
) [static]

Gets the path of a library based on added paths. Paths will be checked in order or priority, then order added, then specificity. For example, if a path is added as such:

Loader::addPath('/app/includes/Some/Path', 'Some', 'app');

And the class is loaded, priority paths will be tested first, followed by app paths and finally core paths. Paths with type Some will be checked first for a file called SomeClass.php. If not found, paths with type Some will be checked for a file called Path/SomeClass.php. Finally, paths with type * will be checked for a file called Some/Path/SomeClass.php

Parameters:
string $class The name of the class to be checked
scope The scope to check. If null, all scopes will be checked.
Returns:
The path to the requested library if it is found, or null.
